# ML Foundations: Regression & Clustering

A collection of machine learning projects demonstrating both supervised and unsupervised learning techniques using Python and Scikit-learn.

This repository showcases the complete machine learning workflow, including data preprocessing, exploratory analysis, model building, evaluation, and result interpretation.

---

## Projects Included

### 1. Student Performance Prediction
A supervised learning project that uses **Linear Regression** to predict student academic performance based on various features.

**Key concepts:**
- Data cleaning and preprocessing
- Handling categorical and numerical features
- Train-test split
- Feature encoding
- Linear Regression
- Model evaluation

Dataset:
- `student_performance_dirty.csv`

Notebook:
- `Linear_regression_model.ipynb`

---

### 2. Customer Segmentation
An unsupervised learning project that applies **K-Means Clustering** to identify customer groups with similar purchasing behaviors.

**Key concepts:**
- Exploratory Data Analysis (EDA)
- Feature selection
- Feature scaling
- Elbow Method
- K-Means Clustering
- Cluster visualization and interpretation

Dataset:
- `online_retail.csv`

Notebook:
- `Cluster_Kmean.ipynb`

## Installation

Clone the repository:

```bash
git clone <repository-url>
cd ML-Foundations-Regression-and-Clustering
```

Install the required packages:

```bash
pip install pandas numpy matplotlib seaborn plotly scikit-learn jupyter
```

Launch Jupyter Notebook:

```bash
jupyter notebook
```

Open either notebook and run all cells.
